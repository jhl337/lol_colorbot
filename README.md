# Questions & Answers
## Where can I get the 'ghostbox'?
    The 'ghostbox' is a hardware to simulate the mouse, you can use kmbox or the other libraries to replace it.
## How can I use it?
    1. Install python.
    2. Install dependencies via pip.
    3. Edit the config and run the .py file.
